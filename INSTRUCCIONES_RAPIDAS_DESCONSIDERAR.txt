PASO 1: MODIFICAR TU EXCEL
═══════════════════════════════════════════════════════════════

1. Abre tu Excel actual (el que tiene los 1180 SKUs)

2. ELIMINA todas las columnas EXCEPTO:
   - SKU

3. AGREGA una nueva columna:
   - ✅ Desconsiderar

   ⚠️ Copia el emoji desde aquí si no lo tienes: ✅

4. Llena la columna "✅ Desconsiderar" con "SI" en TODAS las filas:

   Atajo rápido:
   - Escribe "SI" en la primera celda (B2)
   - Copia esa celda (Ctrl + C)
   - Selecciona desde B2 hasta B1181
   - Pega (Ctrl + V)

5. Tu Excel debe quedar así:

   ┌─────────────┬──────────────────┐
   │ SKU         │ ✅ Desconsiderar │
   ├─────────────┼──────────────────┤
   │ 010918VE    │ SI               │
   │ 01HR3309    │ SI               │
   │ 0           │ SI               │
   │ ...         │ SI               │
   └─────────────┴──────────────────┘

6. IMPORTANTE: Verifica que la hoja se llama "Datos"
   - Si no, haz clic derecho en la pestaña → Cambiar nombre → "Datos"

7. Guarda el archivo: Desconsiderar_1180.xlsx

═══════════════════════════════════════════════════════════════
PASO 2: SUBIR EN EL DASHBOARD
═══════════════════════════════════════════════════════════════

1. Abre navegador: http://localhost:3012/dashboard

2. Busca el botón de upload
   ⚠️ Puede estar en la parte superior del dashboard
   ⚠️ Si dice "Importar Actualizaciones", úsalo (ahora con el Excel correcto funcionará)

3. Click en el botón → Selecciona archivo → Desconsiderar_1180.xlsx

4. Click "Abrir"
